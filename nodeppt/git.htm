Git简介

  • 分布(/集中)式版本管理系统
  • Linux
  • GitHub, fork, pull request, issue
  • Branch
  • tag
  • pod

配置

// 查看配置列表
$ git config -l

// 配置git全局邮箱
$ git config --global user.email "邮箱"

// 配置git全局用户名
$ git config --global user.name "用户名"

本地管理

// 初始化项目--全新的项目
$ git init

// 初始化项目--已存在的项目
$ git clone ssh://xdzhangm@iss110301000305/LBTravel.git

// 提交代码到本地版本库
$ git commit -am "xxxx"

// 查看日志
$ git log

// 获取某个时间点的代码
$ git checkout 3286ea4

// 查看当前branch
$ git branch

.gitignore

$ cat .gitignore
.svn
.svn/*
*svn*
.DS_Store
wc.db
build
build/*

Branch

// 从当前分支创建新的分支
$ git checkout -b develop

// 提交代码
$ git commit -am "....."

// 切换分支
$ git checkout master

// 合并代码
$ git merge develop

SSH登录

// 创建公钥、私钥
$ ssh-keygen -t rsa -C "xxx@isoftstone.com"

// 多ssh管理
$ vim ~/.ssh/config
Host iss
    HostName iss110301000305
    User git
    IdentityFile ~/.ssh/id_rsa
$ ssh-add -l

// 上传公钥
$ cat ~/.ssh/id_rsa.pub

提交到服务器

// 添加远程库
$ git remote add origin ssh://xdzhangm@iss110301000305/YeSanPo.git

// 查看远程库
$ git remote -v

// 提交到远程库
$ git push origin master

从服务器获取更新

// 从远程服务器获取最新代码
$ git fetch origin master

// 从远程服务器获取最新代码，并且合并代码
$ git pull origin master

merge

  • 自动合并
  • 冲突解决-kdiff3
    $ git config --global mergetool.kdiff3.path /Applications/kdiff3.app/Contents/MacOS/kdiff3
$ git config --global merge.tool kdiff3

tags

// 创建tag
$ git tag -a v1.0.0 -m "......."

// 查看tag列表
$ git tag -l

// 提交tag到远程服务器
$ git push origin --tags

// 删除本地tag
$ git tag -d v1.0.0

// 删除远程tag
$ git push origin :refs/tags/v1.0.0
