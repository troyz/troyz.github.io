Mobile跨平台框架

框架 开发 特点
cordova + H5 framework H5开发、H5展示 开发速度快、丰富的native plugins
ionic H5开发、H5展示 同上
Angular2（TS）、webpack
提供了比较丰富的 Mobile UI组件
Vue
个人开发者		 H5开发、H5展示 轻量、易用、文档完善
针对Mobile的组件不是特别丰富
Weex
阿里巴巴		 H5开发、Native展示 一次撰写，多端运行
支持Vue开发，手机上渲染成Native组件
部分功能特性由于 native 特性的关系，会略有不同
React、React Native全家桶
Facebook		 H5开发、Native展示 Learn Once, Write Anywhere
React Native还在快速开发迭代、API经常变动

ionic架构

helloworld

// 安装环境
$ npm install -g ionic cordova

// 创建ionic项目
$ ionic start helloworld --v2

// 安装第三方库
$ npm install

// 浏览器调试
$ ionic serve

// 添加 iOS 支持
$ ionic platform add ios@4.3.0

// 编译iOS工程
$ ionic build ios --prod --release

// 添加 Android 支持
$ ionic platform add android@6.1.0

// 编译Android工程
$ ionic build android --prod --release

// 添加native插件
$ ionic plugin add https://github.com/charleyw/cordova-plugin-alipay.git
